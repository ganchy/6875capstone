# Network Configuration, Automation, Validation, and Rollbacks with Python
ICTN 6875 Capstone

This is a network automation lab aimed at undergraduate ICTN students from their sophomore year or onwards. The instructional material covers the configuration, automation, validation, and rollback of network devices using python. Students will create a closed loop automation system to handle automated network device configuration, verify their changes, and learn how they can be reverted. Netmiko is the primary tool used for gaining CLI access to the CSR1000v over SSH. Students will need to use the course provided CSR1000v and DEVASC virtual machines to complete the lab. Either the instructor or student lab manual may be followed to deploy the automation for the provided VM's. 

**Important:** IP Addresses must be checked as it is likely your instance of the VM has been given a slightly diferent address than that present in the guide. You must use the addresses on your individual VM instance in the scenario that it does not match the instructions.
