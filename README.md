# ICTN 6875 Network Automation Capstone
This repository serves as the location for storing capstone related files.
  - Diagrams and scripts will be added seperately for use outside of the VM scope of the lab.

The instructional material covers the configuration, automation, validation, and rollback of network devices using python. 
Students will need to use the provided CSR1000v and DEVASC VM's to complete the lab. IP Addresses must be checked as it is likely your instance of the VM has been given a slightly diferent address than that present in the guide. You must use the address on your VM instance in the case that it does not match the instructions.
